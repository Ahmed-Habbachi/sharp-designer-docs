# 5. SummaryItems

- **Type**: SummaryItemType enum, possible values
  Sum = 0,
  Min = 1,
  Max = 2,
  Count = 3,
  Average = 4,
  Custom = 5,
  None = 6
- **FieldName**: string, based field for calculation
- **Format**: string, display format
- **AddToColumn**: string, specify the columns where to add the summary item
